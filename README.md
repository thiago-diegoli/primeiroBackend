---

## Instalação 
Execute os seguintes comandos:
```bash
npm init
npm i express
npm i mongodb
npm i dotenv
npm i express-validator
npm i nodemon -D
```
Edite o package.json, com o seguinte conteúdo:
```json
"main": "api/index.js",
  "type": "module",
  "engine": {
    "node": "18.x",
    "npm": "9.x"
  },
```
